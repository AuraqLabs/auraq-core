# Inspirations

Resources and articles that inform the architecture and philosophy of Auraq Core.

---

## Architecture & Build Philosophy

- [Going Buildless](https://mxb.dev/blog/buildless/) — The case for native ES modules and no build step
- [Dynamically Import ESM from CDN](https://daviddalbusco.com/blog/dynamically-import-esm-modules-from-a-cdn/) — Patterns for importing ESM modules at runtime from a CDN
- [Microfrontends](https://modular.js.org/concepts/microfrontends/) — Composing independent frontend modules into a unified UI

## Tooling & Infrastructure

- [esm.sh](https://esm.sh/) — CDN that serves npm packages as native ES modules
